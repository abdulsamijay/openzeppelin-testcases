# openzeppelin-testcases

Test cases for ERC20 made with OpenZeppelin.

Use the following commands to get it working!

1. npm install --save-dev @openzeppelin/cli
2. npm install --save-dev @openzeppelin/test-enviroment mocha chai
3. npm install --save-dev @openzeppelin/test-helpers
4. npm install --save-dev mocha chai
5. npm install web3
